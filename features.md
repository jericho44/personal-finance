# Personal Finance Management Features

## 1. Authentication & User Management
- User Registration
- User Login & Logout
- Password Reset & Recovery
- Two-Factor Authentication (2FA)
- User Profile Management
- Role-Based Access Control
- Session Management
- OAuth Integration (Google/Social Login)

## 2. Dashboard
- Total Balance Overview
- Income vs Expense Summary Chart
- Recent Transactions List
- Budget Progress Warning Alerts
- Category Spending Pie Chart
- Quick Action Buttons (Add Income/Expense)
- Customizable Date Range Filters

## 3. Account Management
- Create Multiple Accounts (Cash, Bank, Credit Card)
- Edit Account Details
- Delete/Archive Accounts
- View Account Balances
- Account Category Types
- Transfer Funds Between Accounts
- Default Currency Selection

## 4. Transaction Management
- Add Income Transaction
- Add Expense Transaction
- Edit/Delete Transactions
- Transaction Categorization
- Add Transaction Tags/Labels
- Date and Time Selection
- Add Transaction Notes/Attachments (Receipts)
- Recurring Transactions Setup
- Search and Filter Transactions

## 5. Categories
- Default Category Templates
- Create Custom Categories
- Edit/Delete Categories
- Sub-Categories Management
- Set Category Colors
- Assign Icons to Categories
- Category Grouping (Fixed vs Variable)

## 6. Budget Management
- Create Monthly/Weekly Budgets
- Edit/Delete Budgets
- Assign Budgets to Categories
- Budget Progress Tracking
- Notification/Alert for Over-budgeting
- Roll-over Budgeting
- Compare Actual vs Planned Budget

## 7. AI-Powered Financial Insights (Google Gemini 2.5 Flash)
- **Intelligent Spending Analysis:** Automated insights into your monthly spending habits.
- **Smart Recommendations:** 3+ actionable tips to save money based on current transactions.
- **Financial Health Score:** A dynamic 1-10 rating of your fiscal wellness.
- **Anomaly Detection:** Flags unusual spending or unexpected spikes.
- **Automated Summaries:** Natural language overviews of your financial status.
